# ✋ Hand Gesture Recognition AI Dashboard

A real-time **AI-powered hand gesture recognition system** built using **TensorFlow, MediaPipe, OpenCV, and Streamlit**.  
This project detects hand gestures from webcam input and converts them into **live predictions + typed text output + analytics dashboard**.

---

## 🚀 Features

- ✋ Real-time hand gesture recognition
- 🎥 Live webcam processing
- 🧠 TensorFlow deep learning model
- 🤖 MediaPipe hand tracking (21 landmarks)
- ⚡ FPS (Frames Per Second) counter
- ⌨️ Gesture-to-text typing system
- 📊 Live analytics dashboard
- 📈 Prediction history tracking
- 📸 Screenshot capture (optional)
- ☁️ Ready for Streamlit Cloud deployment
